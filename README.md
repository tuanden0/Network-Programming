# Network-Programming using C
# HOW TO USE
- Using GCC at least 4.x.x to compile
- Server:
    - gcc -o server server.c
    - ./server
- Client:
    - gcc -o client client.c
    -  ./client localhost

# NOTE
- Some project has lib math.h so you have to add -lm after the command to compile without error.
